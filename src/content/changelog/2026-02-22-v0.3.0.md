# v0.3.0 - Landing SEO + Changelog

## Resumen
En esta version lanzamos la nueva landing principal orientada a SEO y descubrimiento.
El listado completo de metodos ahora vive en `/allMethods`.

## Cambios principales
- Nueva pantalla de inicio con propuesta de valor clara.
- Secciones informativas para explicar que es OSRSTool y como funciona.
- Nuevo sistema de changelog con articulos en markdown.
- Ruta nueva para detalle de cada novedad en `/changelog/:slug`.

## Impacto esperado
Mejor indexacion en buscadores, mejor retencion de usuarios nuevos y una comunicacion mas transparente de cambios de producto.
